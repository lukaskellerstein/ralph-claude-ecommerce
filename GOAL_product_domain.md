# Product Domain — Authentication + Payments demo (mock fixture)

## User Stories

### US-1 — Sign up / log in / log out
An end user visits the site, creates an account, logs in, and logs out cleanly.

### US-2 — Checkout with credit card
A logged-in user places items in a cart and completes checkout.

### US-3 — Close the loop
A third feature exists to exercise multi-cycle checkpoint behavior.

## Scope

In scope: authentication, payments, one follow-up feature.

Out of scope: catalog browsing, inventory, fulfillment, admin panel.
